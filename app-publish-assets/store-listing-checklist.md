## Common Rejection Reasons to Avoid

### Google Play
- [ ] Accurate content rating completed
- [ ] No misleading claims in description
- [ ] Privacy policy accessible and accurate
- [ ] App doesn't crash on startup
- [ ] All features described actually work
- [ ] No copyright violations (icons, content)
- [ ] Permissions justified and explained
- [ ] No deceptive behavior or malware
- [ ] Follows Google Play policies
- [ ] Data safety accurately represents data collection

### Apple App Store
- [ ] App functions as described
- [ ] No placeholder content or unfinished features
- [ ] Privacy policy URL works without login
- [ ] App doesn't request unnecessary permissions
- [ ] No copyright violations
- [ ] Follows Apple Human Interface Guidelines
- [ ] No broken links in app or metadata
- [ ] App provides value and is not just a website wrapper
- [ ] Kids category apps don't include third-party analytics
- [ ] Export compliance answered correctly
